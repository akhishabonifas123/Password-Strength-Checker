Features
Real-time Feedback: Instantly classifies passwords into Weak, Medium, or Strong.
Visual Indicators: Uses color-coded results (Red, Yellow, Green) for better user experience.
Secure Input: Masks password characters as you type.
Multi-Criteria Analysis: Checks for length, casing, numbers, and special characters.

Strength Criteria
The application evaluates passwords based on the following rules:
LengthAt least 8 characters long
CasingIncludes both uppercase and lowercase letters
NumbersContains at least one digit (0-9)
Special CharactersContains symbols (e.g., ! @ # $ % ^ & *)
